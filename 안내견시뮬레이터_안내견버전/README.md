# 🐕 안내견 시뮬레이터 - 나는 안내견!

**안내견 관점**에서 공공장소의 다양한 상황을 경험하는 미니 시뮬레이션 게임입니다.

---

## ✅ 게임 소개

### 🎮 게임 플레이
- **8 스테이지** 퀴즈 (각 1문항, 4지선다)
- **안내견 관점**에서 상황 판단 및 행동 선택
- 정답 +10점 (최대 80점)

### 📚 학습 주제
| 스테이지 | 주제 | 핵심 학습 포인트 |
|----------|------|-----------------|
| STAGE 1 | 나의 역할 | 안내견의 주업무 이해 |
| STAGE 2 | 집중 유지 | 낯선 사람에게 반응하지 않기 |
| STAGE 3 | 간식 거절 | 훈련된 식단 유지의 중요성 |
| STAGE 4 | 신체 접촉 | 쓰다듬기에도 평정심 유지 |
| STAGE 5 | 위험 판단 | 안전을 위한 멈춤 판단 능력 |
| STAGE 6 | 식당 매너 | 공공장소에서 조용히 대기 |
| STAGE 7 | 대중교통 | 최소 공간으로 핸들러 보조 |
| STAGE 8 | 리드 관리 | 핸들러와의 연결 유지 |

---

## 🏆 칭호 시스템

| 점수 | 칭호 |
|------|------|
| 80점 | 완벽한 안내견 🏆 |
| 60~70점 | 믿음직한 파트너 🛡️ |
| 40~50점 | 성장하는 안내견 🔍 |
| 0~30점 | 훈련 중인 강아지 📚 |

---

## 🖼️ 화면 구성

| 화면 | 설명 |
|------|------|
| **인트로** | 게임 소개 + 안내견/핸들러 SVG 일러스트 |
| **게임** | HUD(진행바, 도트, 점수) + 상황 씬 + 선택지 + 피드백 |
| **결과** | 점수 카운트업, 칭호, 정답/오답 통계 |
| **오답 복습** | 틀린 문제만 모아서 복습 |

---

## ✨ 주요 특징

### 연출 효과
- ✅ 정답 시 이모지 파티클 폭발
- ❌ 오답 시 선택지 흔들림(shake) 애니메이션
- 🎯 점수 카운트업 애니메이션
- 🎨 SVG 캐릭터 씬 (안내견, 핸들러, 시민)
- 🏷️ 상황별 배경 라벨 (공공장소/공원/식당/지하철/카페)

### 교육 목표
- 안내견의 역할과 책임 이해
- 일하는 중 집중력 유지의 중요성
- 공공장소에서의 올바른 행동 학습
- 핸들러와의 신뢰 관계 인식

```

  style.css         - 전체 스타일 (변수, 컴포넌트, 애니메이션)
js/
  scenes.js         - SVG 캐릭터 & 씬 빌더
  game.js           - 게임 로직 (문항 데이터, 상태 관리, 렌더링)
```

## 🎯 게임 흐름

1. **인트로 화면** → 게임 소개 및 설명
2. **게임 진행** → 8개 스테이지 순차 진행
3. **결과 화면** → 점수 및 칭호 확인
4. **오답 복습** → 틀린 문제 상세 리뷰 (선택사항)

---

## 🔄 변경 사항 (기존 버전 대비)

### 관점 전환
- **기존**: 일반 시민이 안내견을 만났을 때 매너 학습
- **현재**: 안내견이 되어 다양한 상황에서 올바른 판단 학습

### 문항 재구성
- 모든 8개 스테이지를 안내견 관점으로 완전히 재작성
- 안내견의 역할, 책임, 행동 지침에 초점

### UI/UX 유지
- 기존의 모든 디자인, 애니메이션, 연출 효과 그대로 유지
- 타이틀 및 설명 텍스트만 안내견 관점으로 변경

---

## 💡 사용 방법

1. 웹 브라우저에서 `index.html` 파일 열기
2. "시작하기 🐾" 버튼 클릭
3. 각 상황에서 안내견의 입장에서 가장 적절한 행동 선택
4. 피드백 확인 후 다음 스테이지 진행
5. 결과 화면에서 최종 점수 및 칭호 확인
6. (선택) 오답 복습으로 틀린 문제 다시 학습
