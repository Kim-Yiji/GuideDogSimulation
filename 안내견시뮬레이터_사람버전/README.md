# 🐕 가이드독 매너 시뮬레이터

공공장소에서 안내견을 만났을 때 **"내가 지금 하는 행동이 매너인지 아닌지"** 빠르게 선택하는 미니 시뮬레이션 게임입니다.

---

## ✅ 구현된 기능

### 🎮 게임 플레이
- **8 스테이지** 퀴즈 (각 1문항, 4지선다)
- 각 스테이지: 상황 씬 → 선택지 4개 → 즉시 피드백
- 정답 +10점 (최대 80점)

### 🖼️ 화면 구성
| 화면 | 설명 |
|------|------|
| **인트로** | 게임 소개 + 안내견/핸들러 SVG 일러스트 |
| **게임** | HUD(진행바, 도트, 점수) + 상황 씬 + 선택지 + 피드백 |
| **결과** | 점수 카운트업, 칭호, 정답/오답 통계 |
| **오답 복습** | 틀린 문제만 모아서 복습 |

### 🏆 칭호 시스템
| 점수 | 칭호 |
|------|------|
| 80점 | 매너 마스터 🏆 |
| 60~70점 | 안전 지킴이 🛡️ |
| 40~50점 | 호기심 많은 시민 🔍 |
| 0~30점 | 오늘은 연습이 필요! 📚 |

### ✨ 연출 효과
- 정답 시 이모지 파티클 폭발
- 오답 시 선택지 흔들림(shake) 애니메이션
- 점수 카운트업 애니메이션
- SVG 캐릭터 씬 (안내견, 핸들러, 시민)
- 상황별 배경 라벨 (공공장소/공원/식당/지하철/카페)

## 📁 파일 구조

```
index.html          - 메인 HTML (모든 화면)
css/
  style.css         - 전체 스타일 (변수, 컴포넌트, 애니메이션)
js/
  scenes.js         - SVG 캐릭터 & 씬 빌더
  game.js           - 게임 로직 (문항 데이터, 상태 관리, 렌더링)
```

---

## 🗂️ 문항 목록

| 스테이지 | 주제 | 핵심 학습 포인트 |
|----------|------|-----------------|
| STAGE 1 | 안내견의 정체 | 보행 보조 역할 이해 |
| STAGE 2 | 지금 말 걸어도 될까? | 집중 방해 금지 |
| STAGE 3 | 간식 금지 | 음식/간식 절대 NG |
| STAGE 4 | 식당에서 만났다면 | 공공장소 출입 허용 (법적) |
| STAGE 5 | 도움이 필요해 보일 때 | 핸들러에게 먼저 |
| STAGE 6 | 쓰다듬고 싶은 마음 | 허락 후 + 일 끝난 뒤 |
| STAGE 7 | 지하철에서의 거리감 | 대중교통 탑승 허용 |
| STAGE 8 | 에티켓 최종 보스 | 허락 없는 신체 접촉 NG |
